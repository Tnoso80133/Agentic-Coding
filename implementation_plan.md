# 俄羅斯方塊 (純黑白風格) 實作計畫

這個計畫旨在從零開始建立一個純 HTML、CSS 和 JavaScript 的俄羅斯方塊遊戲，符合純黑白風格（淺色背景）、箭頭控制、音效、預覽、硬下落與旋轉等需求。

## 使用者確認需求 (已確認)

> [!IMPORTANT]
> **按鍵設定確認**
> - **左方向鍵 (←)**：向左移動
> - **右方向鍵 (→)**：向右移動
> - **下方向鍵 (↓)**：軟下落 (加速下落)
> - **上方向鍵 (↑)**：旋轉方塊
> - **空白鍵 (Space)**：硬下落 (瞬間到底並固定)

> [!NOTE]
> **音效實作方式**
> 為了保持「純 HTML」且不需要您額外下載音效檔案，我將使用瀏覽器內建的 Web Audio API (JavaScript 合成器) 來產生復古的 8-bit 風格音效（包含移動、旋轉、硬下落、消除行和遊戲結束的音效）。

## 預計實作變更

### 遊戲核心檔案

將會在 `c:\Users\123\.gemini\Tetris` 目錄下建立以下檔案：

#### [NEW] [index.html](./index.html)
- 包含遊戲主畫布 (`<canvas>`)、下一個方塊預覽區、分數與等級顯示。
- 載入 CSS 與 JavaScript。

#### [NEW] [style.css](./style.css)
- 實作極簡的純黑白風格 (Black & White Theme)。
- 淺色背景，黑色邊框與文字，方塊為黑色（明亮模式，黑線條）。

#### [NEW] [script.js](./script.js)
- **遊戲迴圈 (Game Loop)**：使用 `requestAnimationFrame` 控制下落速度。
- **方塊系統 (Tetrominoes)**：定義 7 種經典方塊及其旋轉矩陣。
- **控制邏輯 (Input Handling)**：監聽鍵盤事件。
- **碰撞偵測與盤面狀態 (Collision & Board)**：10x20 的網格資料結構。
- **Web Audio API**：使用 OscillatorNode 產生各種遊戲音效。

## 驗證計畫

### 手動測試與驗證
1. 使用瀏覽器開啟 `index.html`。
2. 驗證畫面是否為純黑白風格（淺色背景）。
3. 測試方向鍵左、右是否正常移動。
4. 測試上方向鍵是否能正常旋轉方塊。
5. 測試空白鍵（或自訂的鍵）是否能瞬間下落。
6. 確認音效是否在對應操作與事件（如消行）時正確播放。
7. 確認遊戲結束邏輯是否正常運作。
